\section{Side Projects}

\normalsize{A full list of personal side projects available at \href{https://github.com/raymond-w-ko}{github.com/raymond-w-ko} and \href{https://bitbucket.org/rko}{bitbucket.org/rko}.}

\paragraph{\href{https://github.com/raymond-w-ko/omegacomplete.vim}{omegacomplete}}
\normalsize{A keyword completion plugin for the Vim text editor. Uses Python bindings to communicate with Vim and C++ to implement the algorithmic portions. Designed to be multithreaded with background processing to scale with modern CPUs and avoid blocking. Supports various completion types, such as: prefix, title case, camel case, subsequence, closest edit distance, and keywords with underscore and hyphen separators.}

\paragraph{\href{https://github.com/raymond-w-ko/tsukuyomi}{tsukuyomi}}
\normalsize{A Lisp heavily inspired by Clojure which is compiled down to Lua.}

\paragraph{\href{https://bitbucket.org/rko/vim}{JVM bindings for Vim}}
\normalsize{A highly experimental fork of the Vim text editor with JVM (Java Virtual Machine) bindings. This allows JVM hosted languages such as Clojure, Scala, and Groovy to be used inside Vim.}
